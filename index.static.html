Static Page Test

If you can see this page when opening index.static.html directly (double-click), your browser is rendering basic HTML/CSS correctly.

Next steps:

  • Open DevTools (F12) → Console and look for errors when you open index.html.
  • Open DevTools → Network and reload index.html; check for failed resource requests (status ≠ 200).

Quick checks

- If this page displays, the browser renders HTML/CSS fine.
